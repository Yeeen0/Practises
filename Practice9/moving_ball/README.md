# Moving Ball Game

An interactive game with a red ball controlled by arrow keys.

## How to Run

```bash
pip install pygame
python main.py
```

## Controls

| Key        | Action          |
|------------|-----------------|
| ↑ Up       | Move up 20px    |
| ↓ Down     | Move down 20px  |
| ← Left     | Move left 20px  |
| → Right    | Move right 20px |
| ESC        | Quit            |

## Rules
- Ball size: 50×50 pixels (radius 25)
- Each key press moves the ball exactly 20 pixels
- Ball cannot leave the screen — boundary inputs are ignored

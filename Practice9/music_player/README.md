# Music Player

An interactive music player with keyboard controls.

## How to Run

```bash
pip install pygame
python main.py
```

## Controls

| Key | Action         |
|-----|----------------|
| P   | Play           |
| S   | Stop           |
| N   | Next track     |
| B   | Previous track |
| Q   | Quit           |

## Adding Music

Place `.mp3`, `.wav`, or `.ogg` files inside the `music/` folder.
The player will automatically detect and load them.

# Practice 9 — Pygame Games

Three classic games built with Pygame.

## Projects

| Project         | Description                                      |
|-----------------|--------------------------------------------------|
| mickeys_clock   | Clock using Mickey Mouse hands (minutes/seconds) |
| music_player    | Music player with keyboard controls              |
| moving_ball     | Red ball game with arrow-key movement            |

## Setup

```bash
pip install -r requirements.txt
```

## Run

```bash
# Mickey's Clock
cd mickeys_clock && python main.py

# Music Player
cd music_player && python main.py

# Moving Ball
cd moving_ball && python main.py
```
